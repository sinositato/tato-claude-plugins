## FAQ

**Q: Which document should I start with?**
A: If you're new, start with [Quick Reference](./prd-quick-reference.md) for a 10-minute overview, then try examples from [Recommended Prompts](./prd-recommended-prompts.md).

**Q: Do I need to read all 68KB of the Complete Guide?**
A: No. Use it as a reference. Read the sections relevant to your current task.

**Q: How do I know which template to use?**
A: See the decision tree in [Quick Reference § Template Selection](./prd-quick-reference.md#template-selection).

**Q: What's the difference between validate and analyze?**
A: **validate** = fast structural check (1-2 min), **analyze** = comprehensive quality analysis (3-5 min). See [Complete Guide § Individual Skill Reference](./prd-skills-complete-guide.md#individual-skill-reference).

**Q: Where are the recommended prompts?**
A: [Recommended Prompts](./prd-recommended-prompts.md) has 100+ prompts organized by scenario.

**Q: How do I troubleshoot issues?**
A: Check [Complete Guide § Troubleshooting](./prd-skills-complete-guide.md#troubleshooting) or browse [Recommended Prompts § Troubleshooting](./prd-recommended-prompts.md#troubleshooting).
