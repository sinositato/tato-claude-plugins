# PRD Status Skill

Scans the `prd/` directory and produces a dashboard showing all PRDs with their status, health assessment, section coverage, word count, and available artifacts (context files, summaries, tasks). Suggests quick actions for PRDs that need attention.

## Usage

```
/prd-status
/prd-status draft       # Filter by status
/prd-status todo        # Filter by name
```

## Output

Terminal report (read-only, no files modified).
