# PRD Export Skill

Generates a condensed stakeholder-friendly summary from a full PRD. Extracts key sections (Executive Summary, Goals, Scope, Release Plan, Risks, Open Questions) into a 1-2 page markdown document.

## Usage

```
/prd-export <path-to-prd.md>
```

## Output

`prd/<feature-name>/prd-summary.md`
